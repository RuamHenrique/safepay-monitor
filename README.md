# SafePay Monitor

Sistema de monitoramento de fraudes financeiras desenvolvido para demonstrar uma arquitetura completa com front-end, back-end, banco de dados e pipeline de CI/CD.

---

## 🚀 Tecnologias utilizadas

### Backend
- Java 17
- Spring Boot
- Spring Data JPA
- Hibernate
- Maven

### Banco de Dados
- PostgreSQL

### DevOps
- Docker & Docker Compose
- GitHub Actions (CI/CD)

### Frontend
- HTML simples (interface de teste)

---

## 📦 Funcionalidades

- CRUD completo de usuários
- Integração entre frontend e backend
- Persistência de dados em banco PostgreSQL
- Containerização com Docker
- Pipeline de integração contínua (CI)

---
